## Features
Features
* Map view of restauraunts
* Card layout for restauraunts
* User can leave a review
* User can edit a review
* User can delete a review
* User can favorite a restauraunt
* User can un-favorite a restauraunt

## Tools Used
* React
* Redux
* Material-ui
* Mapbox gl
* React-Router
